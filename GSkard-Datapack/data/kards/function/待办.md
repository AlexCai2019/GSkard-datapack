#
仅对 "基本牌" "军团牌" "装备牌"生效
对于基本牌或军团牌
只需要常规的怪物头颅图标即可 标注在卡牌右下角
如深海恐惧 会召唤守卫者 那么在卡牌右下角标注守卫者头像图标
若卡牌种不止1种生物种类则1s进行一次头像图标的变换（如果能做到的话）

对于装备牌
可有可无吧这里的图标
如获取钻石套则标注钻石盔甲
铁剑则标注铁剑
